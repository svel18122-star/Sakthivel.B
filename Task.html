1. Learn what HTML is, its history, and why it is used.


HTML (HyperText Markup Language) is the standard language used to create and structure web pages. It tells the web browser how to display text, images, links, videos, and other content on a webpage.

History of HTML:

  • HTML was created by Tim Berners-Lee in 1991 while working at CERN.
  • The first official version, HTML 2.0, was published in 1995.
  • Over time, newer versions improved features (HTML 3.2, HTML 4.01).
  • In 2014, HTML5 became the latest standard, supporting multimedia (audio, video) and advanced features without plugins.

    • Why HTML is used:

  • To create the structure of web pages.
  • To link documents through hyperlinks (the foundation of the World Wide Web).
  • To embed images, audio, video, and interactive forms.
  • It is the backbone of all websites and works together with CSS (for design) and JavaScript (for functionality).

    3. Explore the root element html. Learn how it wraps the entire document.

  • The html tag is called the root element in HTML.
  • It tells the browser that everything inside it is an HTML document.
  • All the visible (body) and invisible (head) parts of a webpage must be written inside html... .
  • It usually has a lang attribute (example: html lang="en") to define the language of the page for browsers and search engines.

    4. Learn about container elements like div, section. Practice grouping content.


    Container Elements in HTML

  • div (Division)
  • A generic container used to group elements together.
  • Doesn’t have any special meaning; mainly used for layout and styling with CSS.
  • Example: wrapping text, images, or buttons inside a box.

  • Section

  • A semantic container that represents a section of a document (like chapters, topics, or grouped content).
  • Used when the content inside has a related theme or meaning.
  • Helps with accessibility and SEO.
